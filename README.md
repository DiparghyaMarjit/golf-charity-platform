# ⛳ Golf for Good

A full-stack web application that allows users to participate in golf-based draws, track scores, win prizes, and support charities through subscriptions.

---

## 🚀 Live Demo

👉 https://golf-charity-platform-two-inky.vercel.app

---

## 📌 Features

- 🔐 User Authentication (Signup/Login)
- 📊 Dashboard with:
  - Subscription status
  - Total winnings
  - Scores logged
- 🎯 Monthly draw system
- 🏆 Leaderboard
- ❤️ Charity support integration
- 💳 Subscription plans (Monthly/Yearly)
- 📱 Fully responsive UI

---

## 🛠️ Tech Stack

- **Frontend:** Next.js 16 (App Router), React, Tailwind CSS  
- **Backend:** Next.js API Routes  
- **Database & Auth:** Supabase  
- **Deployment:** Vercel
